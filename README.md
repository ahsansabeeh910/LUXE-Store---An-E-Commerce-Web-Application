# 📌 Overview

**LUXE Store** is a modern luxury-themed e-commerce web application designed using pure:

- HTML
- CSS
- JavaScript

The application delivers a premium online shopping experience with:

✅ Dynamic Product Listings  
✅ Advanced Filtering & Search  
✅ Wishlist Management  
✅ Shopping Cart System  
✅ Checkout Simulation  
✅ Responsive User Interface  

The project demonstrates advanced frontend web development concepts with interactive UI/UX design principles.

# ⚙️ Core Functionalities

## 🛒 Shopping Cart System

Users can:

- Add products to cart
- Increase/decrease quantity
- Remove products
- View live cart totals
- Simulate checkout process

---

## ❤️ Wishlist System

Users can:
- Save favorite products
- Remove wishlist items
- Manage liked products dynamically

---

## 🔍 Product Search & Filtering

The application supports:

- Real-time product search
- Category filtering
- Price range filtering
- Rating filtering
- Product sorting

---

## 📦 Product Detail Modal

Each product includes:
- Detailed description
- Ratings & reviews
- Price information
- Size selection
- Color selection
- Add-to-cart actions
